	Naming Conventions:
 
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
		Pointer: 
			Lowercase "p" before name   
---------------------------------------------------------------------------
			Should have a functional name.
    
---------------------------------------------------------------------------
		Command Groups: 
			(name of function)CmdGrp
    
---------------------------------------------------------------------------	
		Commands: 
			(Name of command)CMD
 
 
---------------------------------------------------------------------------     

	Variable Naming Conventions:
    
		Local: 
			m_(Variable name) : Camel Case
    
---------------------------------------------------------------------------	
		Global: 
			(Variabel name) : Camel case
    
---------------------------------------------------------------------------		
		Classes: 
			Pascal Case
    
---------------------------------------------------------------------------
		Literalls: 
			All caps

---------------------------------------------------------------------------
		Functions: 
			Pascal Case
